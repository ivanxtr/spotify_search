## Installation
```
 yarn install
```

# Commands
## Start development server
```
  - yarn start 
  - go to : http://localhost:3000
```

## Pre-production server
```
  - yarn build
  - yarn ssr
  - go to : http://localhost:8001
```
## Create a new build
``` 
  - yarn build
```
## Run unit test
```
  - yarn test
```

## Run E2E test
```
  - yarn start
  - open a new terminal
  - yarn e2e
```
